# React Carousel Collection 

A modern React.js project showcasing **5 different carousel/slider implementations** with responsive layouts, smooth animations, autoplay support, navigation controls, and customizable configurations.

##  Live Demo

https://react-carousel-beryl.vercel.app/

---

##  Features

- 5 unique carousel/slider designs
- Fully responsive UI
- Smooth transitions & animations
- Navigation arrows & pagination
- Autoplay functionality
- Mobile-friendly swipe support
- Reusable React components
- Clean and scalable folder structure

---

##  Technologies Used

- React.js
- JavaScript (ES6+)
- CSS3
- Swiper / React Slick / Custom Slider Logic

---

## 📂 Carousel Types Included

### 1. Basic Image Slider
Simple image carousel with next/previous controls.

### 2. Autoplay Carousel
Automatically sliding carousel with pause-on-hover functionality.

### 3. Multi-Item Carousel
Displays multiple items/cards simultaneously with responsive breakpoints.

### 4. Infinite Loop Slider
Smooth infinite scrolling experience with continuous looping.

### 5. Custom Animated Carousel
Creative animated transitions using custom React state management.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/react-carousel.git
```

Navigate to the project folder:

```bash
cd react-carousel
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

## ▶️ Usage

You can explore different carousel implementations from the homepage and customize them according to your project requirements.

Example component usage:

```jsx
<Carousel
  autoplay={true}
  slidesToShow={3}
  infinite={true}
/>
```
